## Repository Structure

Solutions are organized by topic folder and problem slug:

```
<topic-folder>/
  <problem-id>/
    submission-0.<ext>   ← first submission
    submission-1.<ext>   ← second submission (revised or different language)
    ...
```

**Example:**

```
Data Structures & Algorithms/two-integer-sum/submission-0.py
Data Structures & Algorithms/binary-search/submission-0.py
```
